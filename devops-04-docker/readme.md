## Images verwalten
docker pull [image]         Lädt ein Image aus einer Registry.
docker images               Listet alle lokalen Images auf.
docker rmi [image]          Entfernt ein lokales Image.
docker push [image]         Lädt ein lokales Image in eine Registry hoch.

## Container verwalten
docker run [image]          Startet einen neuen Container aus einem Image.  
docker ps                   Listet alle laufenden Container auf.        
docker ps –a                Listet alle Container auf, einschließlich gestoppter.
docker stop [container]     Stoppt einen laufenden Container.
docker start [container]    Startet einen gestoppten Container.
docker restart [container]  Startet einen Container neu.
docker rm [container]       Entfernt einen gestoppten Container.

## Netzwerke
docker network ls           Listet alle Netzwerke auf
